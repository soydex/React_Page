# My React + Tailwind App

Ce projet est une application web moderne construite avec React, Tailwind CSS et Vite. Il s'agit d'un modèle de démarrage conçu pour les développeurs souhaitant créer des applications rapides, réactives et esthétiques avec un support pour le mode sombre.

## Fonctionnalités principales

- **Mode sombre** : Basé sur les préférences système ou sélection manuelle avec persistance via `localStorage`.
- **Composants réutilisables** : Inclut des composants comme Navbar, Card, Button, et HeroSection.
- **Animations fluides** : Intégration avec Framer Motion pour des animations modernes.
- **Responsive Design** : Optimisé pour les appareils mobiles et les écrans de bureau.
- **Configuration Vite** : Pour un développement rapide avec HMR (Hot Module Replacement).

## Technologies utilisées

- **React** : Bibliothèque JavaScript pour construire des interfaces utilisateur.
- **Tailwind CSS** : Framework CSS utilitaire pour un design rapide et flexible.
- **Vite** : Outil de build moderne pour un développement rapide.
- **Framer Motion** : Bibliothèque pour des animations fluides et interactives.
- **React Icons** : Pour des icônes vectorielles prêtes à l'emploi.
- **Zustand** : Gestion d'état simple et légère.
- **React Hook Form** : Gestion des formulaires avec validation.

## Installation

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/mon-projet-react-tailwind.git
   ```
2. Accédez au dossier du projet :
   ```bash
   cd mon-projet-react-tailwind
   ```
3. Installez les dépendances :
   ```bash
   npm install
   ```
4. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
5. Ouvrez votre navigateur à l'adresse [http://localhost:3000](http://localhost:3000).

## Structure du projet

- `src/components` : Contient les composants réutilisables comme Navbar, Button, etc.
- `src/contexts` : Fournit le contexte pour la gestion du thème (mode clair/sombre).
- `src/assets` : Contient les images et ressources statiques.
- `src/index.css` : Fichier CSS principal avec les styles globaux.
- `vite.config.js` : Configuration de Vite pour le projet.

## Scripts disponibles

- `npm run dev` : Démarre le serveur de développement.
- `npm run build` : Génère une version de production.
- `npm run preview` : Prévisualise la version de production.
- `npm run lint` : Analyse le code pour détecter les erreurs.
